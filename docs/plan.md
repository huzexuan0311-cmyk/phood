# 饭来 - 产品规划

## 定位

从"我有什么食材"出发，解决"今天吃什么"的决策难题。

## 目标用户

- 25-40岁家庭做饭人群
- 每周至少做3顿饭
- 经常纠结"吃什么"

## 核心流程

```
拍照 → 识别食材 → 推荐菜谱 → 缺啥清单 → 一键买菜
```

## 变现路径

| 阶段 | 用户量 | 方式 |
|------|--------|------|
| MVP | 0-1000 | 免费获客 |
| 增长 | 1k-10k | 生鲜电商CPS分佣 |
| 规模 | 10k-100k | 会员制 9.9/月 |
| 成熟 | 100k+ | 自有品牌食材订阅 |

## 竞品差异化

- 下厨房/豆果：搜索驱动 → **我们是识别驱动**
- 美团买菜：有什么卖什么 → **我们是基于你已有的推荐**
- 海外SideChef：不适用中国食材和口味

## 关键指标

- 次日留存 > 40%
- 拍照识别准确率 > 90%
- 菜谱点击率 > 60%
- 购物清单转化率 > 15%
