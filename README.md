# 饭来 🥬

> 拍一拍冰箱，AI 识别食材 → 推荐菜谱 → 一键买菜

## 项目结构

```
饭来/
├── index.html          # 主页面（MVP 原型）
├── src/
│   ├── style.css       # 全部样式
│   ├── app.js          # 核心逻辑
│   └── mockData.js     # 模拟AI数据（3个场景）
├── docs/               # 文档
│   └── plan.md         # 产品规划
└── README.md
```

## 快速开始

直接在浏览器打开 `index.html` 即可体验。

## MVP 功能

- ✅ 拍照/选照片上传
- ✅ AI 识别食材（模拟）
- ✅ 根据食材推荐菜谱
- ✅ 口味偏好筛选（快手菜/减脂/汤羹/硬菜）
- ✅ 菜谱详情 + 做法步骤
- ✅ 购物清单（缺啥补啥）
- ✅ 一键跳转买菜平台
